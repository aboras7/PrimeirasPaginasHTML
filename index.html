Gatinhos do Edu

Olá, seja bem vindo!

Essa é uma página HTML teste, feita com o intuito de criar um projeto básico
em HTML, para conhecer os primeiros comandos e tags dessa linguagem de marcação.
O que apresento aqui são os gatos do criador destas páginas.

Esta página é divida em quatro links: Menu, Bob, Trio e Neguinha.
O link "Menu" retorna esta mesma página.
O link "Bob" retorna a página do gato Bob.
O link "Trio" retorna a página que contém os gatos Belinha, Branquinho e Branquinha.
O link "Neguinha retorna a página da gata Neguinha.
E o link "Contato" retorna a página de contatar o criador da página.

Todos os meus gatos vieram da rua, com exceção do Bob que nasceu aqui em casa.
Sendo assim, a maioria deles tem suas peculiaridades, medos, traumas, resistências.
Mas é isso que os torna únicos!
